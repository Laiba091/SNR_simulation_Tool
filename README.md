# SNR_simulation_Tool
**Audio SNR Simulation Tool**

This tool generates realistic noisy audio by mixing clean speech with background noise at adjustable SNR levels (-40 to +40 dB). It helps simulate different real-world acoustic conditions and evaluate how noise affects speech clarity.
Users can upload audio, set SNR, and instantly hear and visualize the difference between clean, noisy, and final mixed audio in an easy-to-understand format for both technical and non-technical stakeholders.
